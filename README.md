# Documentation for Project Stallion Version Control

## Version 1.0.0
Download data from individual races from Betfair and William Hill.
Data captured every second and stored as CSV files with columns:

  **Betfair**
  1. Horse name
  2. Back multiplier
  3. Lay multiplier
  
  **William Hill**
  1. Horse name
  2. Win multiplier
  3. Place multiplier
